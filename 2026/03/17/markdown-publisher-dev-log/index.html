Markdown多平台发布工具开发记录

2026-03-17 | 项目开发 前端技术

项目简介

最近开始开发一款Markdown多平台发布工具，旨在解决内容创作者跨平台发布的效率问题。目前已完成前端基础框架搭建。

核心功能规划

1. 多平台发布支持

  • 微信公众号：富文本HTML生成
  • 小红书：纯文本/富文本格式转换
  • 知乎：API发布支持

2. AI智能助手

  • 标题自动生成
  • 内容润色优化
  • 敏感词检测
  • 自定义Prompt模板

3. 编辑器功能

  • Markdown即时渲染
  • 代码高亮、表格支持
  • 图片拖拽上传
  • 多平台预览切换

技术选型

层级 技术栈
前端框架 Vue 3 + TypeScript
构建工具 Vite
UI组件库 Element Plus
编辑器 Vditor
状态管理 Pinia
后端框架 NestJS（规划中）
数据库 PostgreSQL（规划中）

项目结构

开发计划

完善编辑器功能和平台预览模块

第二阶段：后端开发

搭建NestJS服务，实现用户系统和数据存储

第三阶段：功能集成

集成AI能力，完善发布流程

第四阶段：测试优化

性能优化，用户体验改进

总结

目前项目处于前端开发阶段，基础框架已经搭建完成。接下来将重点开发平台转换器和AI助手功能，同时准备后端服务的搭建。

这是一个长期项目，计划在接下来的几个月内逐步完善各项功能，最终打造一款实用的内容创作工具。

项目持续开发中，敬请期待后续更新。
